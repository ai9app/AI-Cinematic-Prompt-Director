# 🎬 AI Cinematic Prompt Director 

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![PRs Welcome](https://img.shields.io/badge/PRs-welcome-brightgreen.svg)](http://makeapullrequest.com)

A professional, structured System Prompt framework designed to generate highly precise, commercial-grade video prompts for advanced AI video generation models like **Sora, Kling, Runway Gen-3, Luma Dream Machine, and Seedance 2.0**.

## 🌟 Why Use This Framework?
AI video models require highly specific, visually grounded language to produce consistent results. This framework acts as a "Virtual Cinematographer," transforming vague user ideas into structured prompt matrices by controlling:
- **Camera Movements** (e.g., FPV Tracking, Dolly Zoom)
- **Shot Types** (e.g., Extreme Close-Up, Dutch Angle)
- **Lighting & Vibe** (e.g., Volumetric Lighting, Cyberpunk Neon)
- **Visual Effects (VFX)** (e.g., Anamorphic lens flare, Film grain)

## 🚀 How to Use

### Method 1: Use as a Custom AI Agent (Recommended)
1. Copy the entire contents of [`Cinematic-Prompt-Director-Skill.md`](./Cinematic-Prompt-Director-Skill.md).
2. Paste it into the "System Prompt" or "Instructions" section of your favorite AI Agent builder (e.g., ChatGPT Custom GPT, Coze, Dify, or Claude).
3. Start chatting with your agent by giving it a simple idea like: *"Make a futuristic car commercial in a neon city."*

### Method 2: Manual Prompting
You can read through the knowledge base in the Markdown file to learn professional cinematography vocabulary and manually construct your prompts using the provided matrix structure.
